# Windows-Style-Operating-System
An windows style operating system made using Java Swing.

#Features
 1. File Browsing
 2. Calendar
 3. Time and Date picker
 4. Refresh options
 5. Task Manager
 
 #Technology used
 1. Java
 2. Java Swing
 3. Java Calendar library
 4. Java Date and Time picker library
 5. Java Robo library
